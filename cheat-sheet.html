Array Cheat Sheet

Create New Array

Array.new 

    new_array = Array.new
    new_array # => []

    new_array = Array.new(3) 
    new_array # => [nil, nil, nil] 

    new_array = Array.new(3, "example")
    new_array # => ["example", "example", "example"]
Literal array constructor 
    numbers = []
    numbers # => []

    numbers = [1, 2, 3]
    numbers # => [1, 2, 3]
Adding to an existing array 
      numbers.push(4) #=> [1, 2, 3, 4] 
      numbers << 4 #=> [1, 2, 3, 4]
      numbers.unshift(4) [4, 1, 2, 3]
      numbers.insert(index, item, item, ect)
Removing elements from array 
      numbers.pop => [1, 2]
      numbers.shift => [2, 3]
      numbers.delete(1) => [2, 3]
      numbers.delete_at(1) => [1, 3]
